# $.fn.autoresize()
Autoresize text fields

WARNING: [unstable state]

**Synopsis**

```javascript
$.fn.autoresize( [ Number height ] );
```

**Options**

```javascript
/*
* @param {Number} height - default height
* @this: {jQuery Object}
* @return: {jQuery Object}
*/
```

## Use
*For example you could use the following code:*

*Markup*

```html
<textarea />
```

*Options*

```javascript
$('textarea').autoresize(20);
```
