# .attributes()

Returns an unsorted {object} of the attributes

## Use
*For example you could use the following code:*

**Markup:**

	<div data-attr="foo" class="bar" />

**Call:**

	$('#id').attributes();

**Result:**

	{
		data-attr: 'foo',
		class: 'bar'
	}
