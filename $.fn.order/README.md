# $.fn.order()

Specify the order of elements

## Use

```javascript
$('.a, .b').order([1, 0]);
```

**Before:**

```html
<p class="a" />
<p class="b" />
```

**After:**

```javascript
<p class="b" />
<p class="a" />
```
