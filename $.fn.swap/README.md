# $.fn.swap()

Swap elements

## Use
*For example you could use the following code:*

**Call:**

```javascript
$('.one').swap('.two');
```

**Markup:**

```html
<p class="one">one</p>
<p class="two">two</p>
```

**Result:**

```html
<p class="two">two</p>
<p class="one">one</p>
```
